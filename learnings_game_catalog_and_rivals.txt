Learnings from Game Catalog & Rival Hub Upgrade:

1. Game Catalog Stabilization:
   - Systematic replacement of external links with 'inline' embedMode ensures a seamless, low-latency experience within the PWA.
   - Using internal assets (cipher-surge, halo-ce-web, etc.) as high-fidelity fallbacks for missing WebGL tributes maintains the "Absolute" quality brand.
   - JSON syntax errors (missing commas) often occur in large data files after automated or manual edits; use a python script to fix them systematically.

2. Rival Hub & Social Features:
   - Removing feature gates (like profile completion requirements) improves day-one user engagement.
   - Enhanced search logic (name + genre + status) makes the multiplayer lobby feel more active and discoverable.
   - Professional DM UI requires tracking both sender and recipient IDs to filter specific conversation threads.
   - Expanding streaming platform support (Kick, TikTok, etc.) aligns the app with modern creator culture.
